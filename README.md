## Expectations
Build a fitness community app using HTML, CSS, JS, and Firebase with features like:
- User Authentication
- Profile Management
- Workout Tracking
- Buddy Matching
- BMI & Goal Progress Tracking
- In-app Messaging
- Challenges & Group Workouts
